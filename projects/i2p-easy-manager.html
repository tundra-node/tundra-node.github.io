I2P Easy Manager

v0.1.0-alpha

View on GitHub

Tool to make I2P less annoying to set up. Handles I2Pd installation and Firefox config automatically.

What it does

Installs I2Pd, sets up Firefox proxy, manages tunnels, and gets you connected without touching config files.

Why I built it

I2P setup is way more complicated than it needs to be. Got tired of doing it manually.
